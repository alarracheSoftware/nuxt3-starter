# Nuxt 3 Starter

A frontend starter with nuxt3 and essential plugins.

Plugins installed :
- Vite (builder)
- Vuetify (css)
- Material design icons (icons)
- i18n (internationalization)
- Pinia (Store)
- Axios (HTTP client)

This starter has a default interface that includes :
- App Bar
- Open-close App Menu
- Router
- Dark theme
- Reusable Snack work with Pinia
