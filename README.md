# Nuxt 3 Starter
A frontend starter with nuxt3 and essential plugins.

## Requisites
- node 20

## Plugins installed
- Builder : **Vite**
- CSS : **Vuetify 3**
- Icons : **Material design icons - mdi**
- internationalization : **i18n**
- Store : **Pinia** 

## Default components
- App Bar
- Open-close App Menu
- Router
- Dark theme
- Reusable snack that works with Pinia
- Reusable confirm dialog that works with Pinia
